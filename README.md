# SKYEYE AI - Real-Time Tracking & Prediction System

## Overview
SKYEYE is a real-time AI tracking system that combines computer vision, state estimation, and predictive targeting.

It detects objects(in this version, Human) , tracks their movement, and predicts future positions using Kalman filtering.

## Features
- Real-time object detection (YOLO)
- Target tracking with Kalman Filter
- Lead prediction (future position estimation)
- HUD visualization (crosshair, radar, compass)
- Distance estimation
- Beginner-Friendly

## System Pipeline
Camera Input → Detection → Tracking → Prediction → Visualization

## Tech Stack
- Python
- OpenCV
- YOLO (Ultralytics)
- Kalman Filter

## Key Concept
This project demonstrates a real-time predictive system
